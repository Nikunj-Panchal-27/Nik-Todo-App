# 📝 Nik Todo App (Java)

A powerful **Advanced Todo Application built using Java** that helps users manage daily tasks efficiently.
The application allows users to **create, update, delete, and schedule tasks with priority and reminders**, making productivity management simple and effective.

---

## 🚀 Features

* ✅ Add new tasks easily
* 🗂️ Edit and update existing tasks
* ❌ Delete completed or unwanted tasks
* ⏰ Schedule tasks with **date and time**
* 🔔 Task **notification / reminder support**
* ⭐ Set **task priority levels**
* 📋 Organized task list for better productivity
* 💾 Persistent data storage

---

## 🛠️ Technologies Used

* **Java**
* **Android Studio** *(if Android app)*
* **XML Layouts**
* **AlarmManager**
* **BroadcastReceiver**
* **SQLite / Local Storage**

---

## 📱 Application Screens

* Home Screen – Displays all tasks
* Add Task Screen – Create a new task
* Edit Task Screen – Update existing task
* Reminder / Notification system

2. Open the project in **Android Studio**

3. Build and run the application on an **Android device or emulator**
